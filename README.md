# Haxe NodeLibs

## Overview

Extern type definitions for the most common Node.JS Libraries like:

- `mongodb`
- `express`
- `multiparty`
- `nodewebkit`

Haxe-generated API documentation is available at []

Original node.js documentation can be found at http://nodejs.org/api/index.html.
